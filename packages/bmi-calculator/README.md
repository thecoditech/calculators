# BMI Calculator

> Body Mass Index (BMI) calculator module

## Install

```
$ npm install @infinitetoolbox/bmi-calculator
```

## Usage
```js
import bmiCalculator from '@infinitetoolbox/bmi-calculator';

const bmi = bmiCalculator(mass, height);
```
